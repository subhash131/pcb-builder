The **Standard AI Agent** is the perfect starting point for building autonomous workflows. 

## What's Included?

- **Core Memory**: Persistent storage for agent context.
- **Task Management**: Multi-step planning and execution.
- **Email Integration**: Built-in support for sending and receiving emails.

## Setup Instructions

1. Configure your environment variables.
2. Run the seed command:
   ```bash
   pnpm seed:standard
   ```
3. Start the agent:
   ```bash
   pnpm start:agent
   ```
